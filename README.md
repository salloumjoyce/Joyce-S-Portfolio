# Joyce Salloum — Portfolio

A ready-to-host static site. No build step.

## Deploy on GitHub Pages
1. Create a new GitHub repository.
2. Upload **all files in this folder** (keep them at the repo root — `index.html` must be at the top level).
3. In the repo: **Settings → Pages → Build and deployment → Source: Deploy from a branch**, pick `main` / root, Save.
4. Wait ~1 minute. Your site is live at `https://<username>.github.io/<repo>/`.

## Notes
- Needs an internet connection to load fonts (Google Fonts), the runtime (React), and the embedded campaign videos.
- The `imgslot-*.state.json` files hold the portfolio images — keep them next to `index.html`.
